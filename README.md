Backend/API For Pigaru App
==========================

## Tech Stack
- Nodejs with [express](https://expressjs.com/) 
- [Mongoose](https://mongoosejs.com/)
- MongoDB for database
- Logging with [elasticsearch](https://expressjs.com/) 

## Installation
```bash
$ cp .env.example .env  # edit as required
$ npm install
```

## Run Service
```bash
$ npm run server
```
